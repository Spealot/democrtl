## Easy Demo Control CFG

> **Usage:** type `exec democtrl` or `dc` in console  
> **Restore:** press `x` or type `exec democtrl_restore`

### Controls
*  **Pause/Play:** `Caps Lock`
*  **Fast Forward:** `F1` - `F6`
*  **Slow Motion:** `1` - `6`
*  **Open DemoUI:** `R`
*  **Spec Player:** `E` 
*  **Reset Spec Target:** `0`
*  **Restore Keys:** `x`
